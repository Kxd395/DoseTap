## Concept

A text-free icon centered on DoseTap’s key feature: the valid **dose window**.

- A time ring
- A highlighted arc (valid interval)
- Subtle ticks for time

## Liquid Glass alignment

- Layered icon (background + translucent foregrounds)
- Crisp vector edges
- Alpha in foreground layers to enable embossed/glass depth

Reference: <https://developer.apple.com/design/human-interface-guidelines/app-icons>
